# 数据结构与算法综合实验 - 项目集合

本仓库包含三个基于数据结构与算法的综合实验项目，均采用 C++ 开发，涵盖二叉树、图结构、线性结构的核心应用，实现了从算法设计到工程落地的完整流程。

---

## 项目一：Huffman 图片压缩与解压工具

### 项目简介

基于哈夫曼（Huffman）算法的无损图片压缩工具，支持 BMP 格式图片的压缩与解压，通过前缀编码特性实现高效数据压缩，压缩比率达 16.93%。

### 核心功能

- 读取 BMP 图片文件，统计 256 种字节的出现权值

- 构建哈夫曼树并生成最优前缀编码

- 将图片文件压缩为 `.huf` 格式，保留解压所需关键信息

- 解析 `.huf` 压缩文件，无损还原原始 BMP 图片

- 支持异常处理（空文件、非法路径、非格式文件）

### 环境要求

- 操作系统：Windows 11

- 开发工具：Microsoft Visual Studio 2022

- 依赖：无第三方库，仅使用 C++ 标准库

### 使用说明

1. 编译项目：打开 `HuffmanCompress.sln`，编译生成可执行文件

2. 压缩操作：

    - 运行程序，选择操作 1（压缩）

    - 输入 BMP 文件完整路径（如 `D:\Pic.bmp`）

    - 压缩文件生成于同目录，命名为 `Pic.bmp.huf`

3. 解压操作：

    - 运行程序，选择操作 2（解压）

    - 输入 `.huf` 文件完整路径（如 `D:\Pic.bmp.huf`）

    - 解压文件生成于同目录，还原为原始 BMP 图片
